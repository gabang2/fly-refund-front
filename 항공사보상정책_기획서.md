## 1. 공통 전제 조건

모든 규정에 앞서 아래 전제가 성립해야 보상을 청구할 수 있다.

- **유효한 예약 확인:** 승객이 예약 확인서(PNR)를 보유하고 있어야 함
- **탑승 시도:** 승객이 해당 항공편 탑승 수속을 위해 정해진 시간 내에 체크인을 시도했어야 함
- **패키지 여행 제외:** 패키지 여행(Package Tour) 구성 항공편은 별도 여행자보호법 적용 (항공사 직접 보상과 다름)

## 2. 관할권 판단 기준

하나의 항공편에 여러 국가의 법이 중복 적용될 수 있다. 아래 순서로 판단한다.

### 2.1 판단 우선순위

```
① 출발지 국가가 EU/EEA/UK인가?
   → YES: 항공사 국적 무관하게 EU261 또는 UK261 적용
   → NO: ②로 진행

② 출발지가 제3국이고, 도착지가 EU/EEA/UK이며, 항공사가 EU/UK 국적인가?
   → YES: EU261 또는 UK261 적용
   → NO: ③으로 진행

③ 출발지 또는 도착지가 캐나다인가?
   → YES: APPR 적용
   → NO: ④로 진행

④ 그 외 국가별 규정 적용 (인도, 사우디, 베트남, 브라질, 한국, 미국 순)

⑤ 어느 규정도 해당 없으면: 몬트리올 협약 실비청구 안내
```

### 2.2 관할권 중첩 처리 방침

**영국 ↔ EU 노선에서 EU 국적 항공사 운항 시:** UK261(파운드)과 EU261(유로) 모두 산출하여, 환율 기준 승객에게 더 유리한 조항을 우선 표시한다. 승객이 직접 선택할 수도 있도록 두 결과 모두 제공한다.

## 3. EU261 / UK261 — 유럽 및 영국

### 3.1 적용 범위


| 조건                 | EU261      | UK261        |
| ------------------ | ---------- | ------------ |
| EU/EEA 출발 → 어디든    | 모든 항공사     | -            |
| 제3국 출발 → EU/EEA 도착 | EU 국적 항공사만 | -            |
| UK 출발 → 어디든        | -          | 모든 항공사       |
| 제3국 출발 → UK 도착     | -          | UK/EU 국적 항공사 |


**EU 회원국 기준:** 27개 EU 회원국 + 아이슬란드(IS), 노르웨이(NO), 스위스(CH), 과들루프, 마르티니크, 레위니옹, 마요트, 프랑스령 기아나, 아조레스, 마데이라, 카나리아 제도

### 3.2 보상 대상 사건

**A. 항공편 취소**

- 취소 통보 시점에 따라 보상 여부 결정 (→ 3.4 참고)

**B. 3시간 이상 지연 도착**

- 이륙 지연이 아닌, 최종 목적지 도착 기준 3시간 이상 지연

**C. 비자발적 탑승 거부 (오버부킹)**

- 항공사가 항공권 소지자를 탑승시키지 못한 경우 (자발적 탑승 포기는 제외)

### 3.3 보상 금액 매트릭스


| 비행 거리                   | EU261 (EUR) | UK261 (GBP) |
| ----------------------- | ----------- | ----------- |
| 1,500km 이하              | €250        | £220        |
| 1,500km 초과 ~ 3,500km 이하 | €400        | £350        |
| 3,500km 초과              | €600        | £520        |


**50% 감액 조건 (대체편 제공 시):** 항공사가 대체 항공편을 제공하고 해당 대체편이 아래 기준 이내로 도착한 경우, 위 금액의 50%만 지급.


| 거리            | 50% 감액 적용 기준 (대체편 도착 지연) |
| ------------- | ------------------------ |
| 1,500km 이하    | 2시간 이내                   |
| 1,500~3,500km | 3시간 이내                   |
| 3,500km 초과    | 4시간 이내                   |


### 3.4 취소 시 통보 시점별 보상 여부


| 취소 통보 시점   | 보상 여부       | 조건                                       |
| ---------- | ----------- | ---------------------------------------- |
| 출발 14일 이전  | **보상 없음**   | 항공권 환불/재예약 권리만 있음                        |
| 출발 7~14일 전 | 보상 없음 (조건부) | 대체편이 원출발 -2시간 이내 출발 + 원도착 +4시간 이내 도착인 경우 |
| 출발 7일 미만   | 보상 없음 (조건부) | 대체편이 원출발 -1시간 이내 출발 + 원도착 +2시간 이내 도착인 경우 |
| 위 조건 미충족   | **전액 보상**   | 거리 기준 매트릭스 적용                            |


### 3.5 면책 사유 (비정상적 상황, Extraordinary Circumstances)

**면책 인정 (보상 없음):**

- 기상 악화 (태풍, 폭설, 폭풍 등 극단적 기상)
- 항공 교통 관제(ATC) 파업
- 정치적 소요, 테러 위험, 공항 폐쇄
- 화산재 분출
- 조류 충돌(Bird strike) — ECJ 판례상 면책 인정

**면책 불인정 (보상 대상):**

- 기체 기술결함 / 일상적 정비 문제 (ECJ Wallentin-Hermann 판결)
- 조종사·승무원 갑작스러운 결근
- 항공사 내부 직원 파업 (자사 파업은 면책 불가)
- 지상 조업 차량 충돌 (ECJ Siewert 판결)

> **기획 메모:** "기술결함"을 사용자가 입력하면 보상 가능으로 분류. "날씨"를 입력하면 보상 불가이나 돌봄의 권리(식사/숙박) 청구 가능 안내 필수.

## 4. 캐나다 APPR

### 4.1 적용 범위

캐나다 내 운항 항공편, 캐나다 출발 항공편, 캐나다 도착 항공편 전체.

### 4.2 항공사 규모 분류


| 구분        | 기준                     | 해당 항공사 예시                                                                                      |
| --------- | ---------------------- | ---------------------------------------------------------------------------------------------- |
| 대형(Large) | 직전 2년 연평균 200만 명 이상 수송 | Air Canada, WestJet, Air Transat, Delta, United, American, Air France, Lufthansa, Korean Air 등 |
| 소형(Small) | 200만 명 미만              | 나머지                                                                                            |


### 4.3 보상 매트릭스

**적용 조건:** 항공사 귀책으로 인한 취소 또는 지연, 최종 목적지 도착 기준.


| 도착 지연 시간        | 대형 항공사 (CAD) | 소형 항공사 (CAD) |
| --------------- | ------------ | ------------ |
| 3시간 이상 ~ 6시간 미만 | $400         | $125         |
| 6시간 이상 ~ 9시간 미만 | $700         | $250         |
| 9시간 이상          | $1,000       | $500         |


**추가:** 취소 후 환불을 선택하더라도 항공사 귀책이면 위 표의 최저 구간 보상금($400/$125)을 별도 청구 가능.

### 4.4 면책 사유 (2025년 이후 신 APPR 기준)

2025년 시행 개정 APPR은 "안전을 위한 예외"를 폐지하고, 명시된 예외적 상황이 아니면 모두 보상 대상으로 추정.

**면책 인정:** 기상 악화, 보안 위협, 공항 운영 중단 등 항공사 통제 불가 사유.

> **기획 메모:** 항공편 날짜가 2024년 이하면 구 APPR(안전 예외 조항 존재), 2025년 이상이면 신 APPR(안전 예외 폐지) 적용. 날짜 입력이 필수.

## 5. 미국 US DOT

### 5.1 기본 철학

미국은 EU261 같은 **징벌적 고정 보상금이 없다.** 대신 항공사가 중대한 변경을 한 경우 즉각적인 전액 환불을 보장한다.

유일한 예외: 오버부킹으로 인한 **비자발적 탑승 거부**는 별도 보상 발생.

### 5.2 전액 환불 요건 (2024년 10월 발효)

아래 조건 중 하나라도 해당하면 승객은 전액 환불을 요구할 수 있다. (대체편/크레딧 거부 시)


| 조건          | 국내선    | 국제선    |
| ----------- | ------ | ------ |
| 출발/도착 시간 변경 | 3시간 이상 | 6시간 이상 |
| 공항 변경       | 해당 시   | 해당 시   |
| 경유지 추가      | 해당 시   | 해당 시   |
| 좌석 등급 강등    | 해당 시   | 해당 시   |
| 장애인 접근성 악화  | 해당 시   | 해당 시   |


### 5.3 환불 처리 기준


| 결제 수단 | 환불 기한        |
| ----- | ------------ |
| 신용카드  | 영업일 기준 7일 이내 |
| 현금/기타 | 역일 기준 20일 이내 |


- 승객의 동의 없이 크레딧/바우처로 대체 지급 불가
- 수하물 수수료, 좌석 지정비 등 부가 서비스도 자동 환불 대상

### 5.4 비자발적 탑승 거부 보상

오버부킹으로 인한 비자발적 탑승 거부만 해당 (자발적 포기는 항공사와 협상):


| 대체편 도착 지연                 | 보상금 (티켓 가격 기준) | 상한선    |
| ------------------------- | -------------- | ------ |
| 1~~4시간 (국내) / 1~~4시간 (국제) | 200%           | $775   |
| 4시간 초과 (국내) / 4시간 초과 (국제) | 400%           | $1,550 |


> **기획 메모:** US 관련 항공편에서는 "보상금" 대신 "환불 가능 여부"를 결과로 제공. "3시간 이상 지연이면 전액 환불 요구 가능"으로 안내.

## 6. 대한민국 소비자분쟁해결기준

### 6.1 법적 성격 (중요)

한국 규정은 공정거래위원회 고시 기준으로, **강제력이 없는 행정 가이드라인**이다. 항공사가 거부하면 한국소비자원에 조정 신청을 하거나 소액 심판을 통해 청구해야 한다.

> **반드시 UI에서 명시:** "이 기준은 항공사에게 법적 강제력이 없습니다. 항공사가 거부할 경우 한국소비자원(1372) 또는 소액 심판 신청이 가능합니다."

### 6.2 국제선 지연 보상


| 지연 시간            | 보상 기준   |
| ---------------- | ------- |
| 2시간 이상 ~ 4시간 미만  | 운임의 10% |
| 4시간 이상 ~ 12시간 미만 | 운임의 20% |
| 12시간 이상          | 운임의 30% |


### 6.3 국제선 취소 보상


| 상황                | 보상 기준               |
| ----------------- | ------------------- |
| 대체편 미제공 취소        | 운임 환급 + 최고 $600 USD |
| 출발 1~4시간 전 취소 통보  | 운임 환급 + 운임의 10%     |
| 출발 4시간 이상 전 취소 통보 | 운임 환급만              |


### 6.4 국내선 지연/취소 보상


| 지연 시간        | 보상 기준           |
| ------------ | --------------- |
| 2시간 이상       | 운임의 10%         |
| 3시간 이상       | 운임의 20%         |
| 취소 (대체편 미제공) | 운임 환급 + 운임의 10% |

## 7. 사우디아라비아 GACA

### 7.1 적용 범위

사우디아라비아 출발 모든 항공편 + 사우디 국적 항공사(Saudia, Flynas 등)의 도착편.

### 7.2 지연 보상 (SDR 기준)


| 지연 시간           | 보상금                         |
| --------------- | --------------------------- |
| 3시간 이상 ~ 6시간 미만 | 50 SDR (약 €61 / ₩93,000)    |
| 6시간 이상          | 150 SDR (약 €183 / ₩280,000) |


*SDR 환율은 IMF 일일 기준으로 변동*

### 7.3 취소 보상 (티켓 가치 % 기반)

취소 통보 시점에 따라 **티켓 환불과 별도로** 추가 페널티를 지급받는다.


| 취소 통보 시점     | 추가 페널티      |
| ------------ | ----------- |
| 14일 ~ 60일 전  | 티켓 가치의 50%  |
| 24시간 ~ 14일 전 | 티켓 가치의 75%  |
| 24시간 이내      | 티켓 가치의 150% |


### 7.4 오버부킹 탑승거부

티켓 환불 + 티켓 가치의 **200%** 즉시 지급.

> **기획 메모:** 사우디 규정은 티켓 가격을 입력받아야 계산 가능. 미입력 시 "티켓 금액의 X% 청구 가능"으로 안내.

## 8. 인도 DGCA

### 8.1 적용 범위

인도 내 운항 국내선 항공편 (CAR Section 3, Series M, Part IV 기준).

### 8.2 특징: 거리가 아닌 블록 타임(Block Time) 기준

인도 규정은 비행 거리가 아니라 **예정된 총 비행 시간(게이트-투-게이트)**을 기준으로 보상액을 산정한다.

### 8.3 지연 보상 매트릭스


| 예정 블록 타임 | 실제 도착 지연        | 보상금 (INR)  |
| -------- | --------------- | ---------- |
| 2시간 이하   | 2시간 이상 ~ 3시간 미만 | 최대 ₹5,000  |
| 2시간 이하   | 3시간 이상          | 최대 ₹10,000 |
| 2~5시간    | 3시간 이상 ~ 6시간 미만 | 최대 ₹7,500  |
| 5시간 초과   | 6시간 이상          | 최대 ₹20,000 |


### 8.4 취소 보상

## 10. 브라질 ANAC Resolution 400

### 10.1 적용 범위

브라질 공항 출발 국내선 및 국제선.

### 10.2 기본 원칙

브라질에는 EU261 같은 고정 현금 보상금이 없다. 대신 단계별 물질적 지원 의무가 있다.


| 지연 시간  | 항공사 의무               |
| ------ | -------------------- |
| 1시간 이상 | 통신 수단 제공 (전화, 인터넷 등) |
| 2시간 이상 | 식사 바우처 제공            |
| 4시간 이상 | 호텔 숙박 + 공항↔호텔 교통편    |


4시간 이상 지연 또는 취소 시: **전액 환불 또는 타 항공사 포함 대체 항공편** 제공 의무.

### 10.3 탑승거부 보상 (유일한 고정 보상)


| 노선  | 보상금              |
| --- | ---------------- |
| 국내선 | 250 SDR (약 €310) |
| 국제선 | 500 SDR (약 €620) |


### 10.4 민사 소송 (위자료 청구)

브라질에서는 항공사 귀책으로 여행을 망친 경우, 일반 소비자법(Consumer Protection Code)으로 소액 심판에서 위자료(Moral Damages)를 청구할 수 있으며 통상 최대 $1,700 USD 수준의 판결이 내려지는 사례가 많다.

> **기획 메모:** 브라질 노선에서 보상 불가 결과가 나왔을 때, 소비자법 기반 소송 가능 여부 안내 링크 제공.

## 11. 몬트리올 협약 (기타 국가 공통 기준)

### 11.1 적용 시점

어떤 특정 국가 규정도 적용되지 않는 국제선 항공편에 대해 기본 적용.

### 11.2 핵심 특징

- **고정 보상금 없음:** 실제로 발생한 금전적 손해(식비, 숙박비, 환승 재구매 비용 등)를 영수증으로 입증하여 청구해야 함
- **책임 한도 (2024년 12월 28일 이후 기준):**


| 항목                 | 한도                   |
| ------------------ | -------------------- |
| 항공편 지연으로 인한 재산상 손해 | 6,303 SDR (약 $8,400) |
| 수하물 파손/분실/지연       | 1,519 SDR (약 $2,000) |


- **청구 기한:** 항공편 출발일로부터 2년

### 11.3 UI 안내 방침

몬트리올 협약 해당 항공편에는 다음 메시지 표시:

> "이 항공편은 자동 현금 보상이 아닌 **실비 청구** 방식이 적용됩니다. 지연으로 발생한 호텔비, 식비, 교통비 등의 영수증을 보관하시고 항공사에 청구하세요. 청구 기한은 출발일로부터 **2년**입니다."

## 13. 돌봄의 권리 (Right to Care)

EU261 기준. 보상금 여부와 무관하게 아래 권리는 항상 보장된다.


| 대기 시간           | 항공사 의무               |
| --------------- | -------------------- |
| 2시간 이상 대기 (단거리) | 식사 및 다과, 통신 수단       |
| 3시간 이상 대기 (중거리) | 식사 및 다과, 통신 수단       |
| 4시간 이상 대기 (장거리) | 식사 및 다과, 통신 수단       |
| 다음날로 넘어가는 야간 대기 | 호텔 숙박 + 공항↔호텔 왕복 교통편 |


> **기획 메모:** 기상 악화 등 비정상적 상황으로 보상금이 0원이더라도 돌봄의 권리는 반드시 안내해야 함. 결과 화면에서 별도 섹션으로 표시.
